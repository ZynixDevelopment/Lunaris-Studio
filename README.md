## 2. Description du service

Lunaris Bot est un bot Discord développé par **Lunaris Studio** offrant les fonctionnalités suivantes :

- Système de tickets et de support client
- Outils de modération (ban, kick, timeout, avertissements)
- Système de bienvenue et d'onboarding
- Panneaux de rôles automatiques
- Système de vérification des membres
- Journalisation des événements serveur
- Commandes utilitaires et d'information

Le Bot est fourni tel quel, dans le cadre de services de développement Discord proposés par Lunaris Studio.
